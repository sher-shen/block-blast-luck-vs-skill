# GOAL — Block Blast 8×8：把它在数学上吃透

> 权威目标文件（2026-06-01 与用户对齐后升级为**双目标**）。记忆/旧 project_goal 的数字以本文件为准。

## 一句话目标
把 8×8 Block Blast 这类**单人 + 随机发块 + 生存型**游戏在数学上吃透——**既造出最强的可玩策略**（不靠预知未来、接近最优），**又量化清楚胜负里有多少是能练的技能、多少是练不掉的牌运**。

两面是同一个 MDP 的一体两面：
- **怎么打到最高分** = 求最优策略（最优期望分）→ 这是"分子"。
- **几分靠运气** = 最优策略下仍消不掉的分数波动 = **EVPI（信息价值 / 后见之明的价值）** → 这是用分子算出来的副产品。

## 两条目标线（co-equal，不再是"研究为主、强玩家为辅"）

**目标线 ①（实用）· 最强可玩策略**：学出一个接近最优、不靠预知未来的 8×8 价值函数（②a）。它回答用户的真实动机——"我玩的时候怎么得更高分"——并且能反过来提炼出人类可用的启发式。

**目标线 ②（认知）· 运气/技能分解**：用三条独立证据（方差分解 / 技能阶梯 / 搜索收敛）+ oracle 双通道把"运气"操作化为可计算量：
- **通道 A · 存活运气** ≈ 极小：完美前瞻下几乎不死（hazard 点估计 7×10⁻⁵/轮，单侧 95% Poisson 上界 3.3×10⁻⁴/轮）→ "死"基本是可避免的技术问题（技能），不可约存活运气只剩罕见 Burgiel 杀手序列。
- **通道 B · 计分运气 (EVPI)** ≈ **57–69%**（T=40/50/60，per-seed paired bootstrap 95% CI，非 ratio-of-means）：知道真实未来的玩家相对最强在线玩家的得分优势里，约六成是纯"后见之明价值"（牌运）。

> headline：**运气 = 后见之明的价值 (value of hindsight / EVPI)**；把 "luck is a lack of information" 操作化、算出来。
> ⚠️ 57–69% 现在载重于"strong(beam,无前瞻)=真在线天花板"这一假设 —— **目标线 ① 的 8×8 RL 正是来独立验证它**：两条线在 ②a 汇合。

## 非目标（明确不做）
- ❌ **不再训一个"无分析的通用 DQN 通关器"**。造强玩家是要的，但任何强玩家都必须绑回天花板/运气分析（避开弱 agent 陷阱）。打平 strong 是好结果，不是失败。
- ❌ **不把 8×8 的 57–69% 当精确常数**。它是"≈"、是相对 strong 强度的下界、依赖计分模型与均匀发牌。
- ❌ **不追求通用游戏 luck/skill 理论**。范围锁死 Block Blast 类（单人/随机发块/生存型/非线性计分）。
- ❌ **不为 deadline 牺牲诚实度**。宁可报 inconclusive，不 clip、不挑易种子、不当 headline。
